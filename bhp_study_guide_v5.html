Prerequisites

BEFORE YOU BEGIN
⛔ Required

Python 3 basics — variables, loops, functions, classes, file I/O, installing packages with pip.

⚠ Recommended

Basic networking — TCP/IP, OSI model, sockets, ports, HTTP, DNS. Knowing Wireshark is a major plus.

✓ You Already Have

As an ML engineer: Python mastery, data pipelines, API design, DB skills — you're ahead of 90% of beginners.

⚙ Lab Setup
  • Run all exercises in an isolated VM lab — never on live networks you don't own or have written permission for.
  • Install Kali Linux (attacker) + Windows 10 VM (target) using VirtualBox or VMware Workstation.
  • Use Python 3.10+ and a dedicated venv — keep one GitHub repo to track every script you write.
  • Your ML background means you can build Jupyter notebooks alongside scripts for richer experimentation and notes.
