PHASE_01 // MONTHS 1–3

Foundations of Production ML

Build Reproducibility & Basic Deployment — 10–12 hrs/week

If you can clone your own repo six months from now and reproduce a result in under 5 minutes, you've passed this phase.

[T3.2]Advanced RAG Patterns DOCS ↗
// LEARN
PATTERN HyDE (Hypothetical Document Embeddings) — generate a hypothetical answer, embed it, search. Dramatically improves sparse queries
