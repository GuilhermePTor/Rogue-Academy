WHAT IS ROGUE ACADEMY?

Rogue Academy is a self-directed AI learning initiative ‚ÄĒ built in the open, from the ground up. It started as a personal project: a way to learn the foundational knowledge that actually underlies AI and deep learning ‚ÄĒ not surface-level tutorials, not API wrappers, but the real mathematical and physical intuitions that make these systems make sense.

In its first form, Rogue Academy is a website for curated AI learning resources ‚ÄĒ a hand-picked collection of the best freely available material on the web, organized to take you from first principles to the frontier. No paywalls. No fluff. No filler.

Eventually it will grow into a platform that helps others explore specialized AI topics and contribute meaningfully to the field.

CORE DIRECTIVES

Free First

Every resource listed is freely accessible. Knowledge should not be gated by ability to pay.

First Principles

Skip the wrappers. Learn the math, the physics, the geometry underneath ‚ÄĒ before the frameworks.

Curated, Not Aggregated

Not a link dump. Every resource is chosen for signal density and depth of intuition.

Open Source

Built in public. Fork it, improve it, add to it. The curriculum should evolve with the field.
